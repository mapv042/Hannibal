================================================================================
PROJECT OVERVIEW
================================================================================
A complete Next.js 14 frontend dashboard for the Hannibal WhatsApp appointment
assistant, designed for doctors and healthcare professionals to manage their
patient appointments, availability, and bot settings.

Technology: Next.js 14 | React 18 | TypeScript | Tailwind CSS | Supabase
Total Files: 34 | TypeScript Lines: 2,500+ | Size: 196 KB

================================================================================
QUICK START
================================================================================
1. npm install
2. cp .env.local.example .env.local
3. Edit .env.local with Supabase credentials
4. npm run dev
5. Open http://localhost:3000

For detailed setup, see SETUP.md

SUPPORT & TROUBLESHOOTING
================================================================================

Installation Issues:
  → See SETUP.md "Quick Start" section
  → Check Node.js version (18+ required)
  → Clear npm cache: npm cache clean --force

Configuration Issues:
  → Copy .env.local.example to .env.local
  → Verify Supabase credentials
  → Check NEXT_PUBLIC_API_URL is correct

Runtime Issues:
  → Check browser console for errors
  → Review server logs: npm run dev
  → See SETUP.md Troubleshooting section

API Integration Issues:
  → Verify backend is running
  → Check CORS configuration
  → Test JWT token in Authorization header
